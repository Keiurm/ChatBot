## 概要
- このWebアプリはOpenAIのAPIを用いたチャットボットです。
- モデルは **o1-preview**、**o1-mini**、**gpt-4o**、**gpt4o-mini** から選択可能で、会話途中でも変更可能です。
- OpenAIのAPIは履歴が保存されませんが、過去の会話履歴を保持するように設計しています。
